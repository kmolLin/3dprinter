原始碼

檔案樹

檔案架構

  1. index.html：首頁呼叫分支程式和頁面。
  2. css：存放自訂外觀。
  3. img：存放圖片。
  4. js：存放 javascript 指令。
  5. lib：存放外部套件，有 Angular Fire、Fire Base、Ionic、Ionic UI Router，使用 bower 安裝和更新。
  6. templates：存放 html 頁面檔。

登入畫面

登入畫面會在登入成功或是註冊成功時顯示使用者信箱帳號名稱。
這時使用者可以透過側邊欄來切換聊天室。

大廳畫面

大廳在未登入時可以查看留言，但是無法發言。
登入後可以在公共區域留言。

房間畫面

一般房間在未登入時不能查看留言，無法發言。
登入後可以在分類的區域留言。

Java Script

程式部份：

  1. app.js：主程式，包含登入、登出、發送、讀取的函式。
  2. controllers.js：控制器，當進行連接 DOM 時執行函式，不過這裡沒做設定。
  3. routes.js：路由，編輯並定義每個頁面的位址資訊。
